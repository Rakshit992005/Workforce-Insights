# 🚀 ReactJS Employee Dashboard Assignment

A modern ReactJS application built as part of an internship assignment.  
This application demonstrates authentication, API integration, routing, camera access, data visualization, and UI/UX design.

---

## 📌 Project Overview

This project is a 4-screen ReactJS application that:

- Authenticates a user via a login screen
- Fetches employee data from a REST API
- Displays employee list with pagination
- Shows detailed employee information
- Captures photo using device camera
- Displays captured image
- Visualizes salary data using charts
- Displays employee cities on a map
